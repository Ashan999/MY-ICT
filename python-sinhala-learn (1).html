👨‍👩‍👧 Inheritance — "දෙමාපිය Class"
එකොළොස්වෙනි පාඩම · Advanced OOP!
🧬 Inheritance = Properties inherit!

Parent class ලෙසින් properties සහ methods child class ට inherit කරගන්නවා! Animal → Dog, Cat

class Animal: def __init__(self, name): self.name = name def eat(self): print(f"{self.name} is eating 🍖") class Dog(Animal): # inherits Animal def bark(self): print(f"{self.name}: Woof! 🐕") class Cat(Animal): # inherits Animal def meow(self): print(f"{self.name}: Meow! 🐱") dog = Dog("Rex") cat = Cat("Luna") dog.eat() # Animal method! dog.bark() # Dog method! cat.meow()
Rex is eating 🍖
Rex: Woof! 🐕
Luna: Meow! 🐱

class BankAccount: def __init__(self, owner, balance=0): self.owner = owner self.balance = balance self.history = [] def deposit(self, amount): if amount > 0: self.balance += amount self.history.append(f"+{amount}") print(f"Deposited Rs.{amount}") def withdraw(self, amount): try: if amount > self.balance: raise ValueError("Insufficient!") self.balance -= amount except ValueError as e: print(f"Error: {e}") def statement(self): print(f"Balance: Rs.{self.balance}") for h in self.history: print(" ", h) acc = BankAccount("Kasun", 1000) acc.deposit(500) acc.withdraw(200) acc.statement()
