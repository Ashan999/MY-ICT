The Word / Docs Interface

The toolbar across the top containing all commands. Organised into tabs: Home, Insert, Design, Layout, Review, View. Everything is here ‚Äî no need to memorise menus!

The strip at the very bottom of the window. Shows your current page, total pages, word count, and language. Right-click it to add/remove info.

Text Basics
‚å®Ô∏è
Typing
Click anywhere on the white page area and start typing. The blinking cursor shows where your text will appear. Use Enter for a new paragraph, Shift+Enter for a soft line break.

Backspace vs Delete
Backspace deletes the character to the LEFT of your cursor. Delete removes the character to the RIGHT. Use them together to edit text without retyping everything!
ü•á
Golden Rule
SELECT first, then CHANGE. Highlight text before applying any formatting. This applies to font, size, colour, bold ‚Äî everything!
Saving Your Work
üíæ
Save vs Save As
Save Ctrl+S ‚Äî updates the current file instantly. Save As F12 ‚Äî lets you choose the filename, location, and file type. Always use Save As for a new document first to name it properly.
üìï
Export as PDF
Go to File ‚Üí Export ‚Üí Create PDF/XPS. PDFs cannot be easily edited and look identical on any device ‚Äî perfect for printing or sharing professionally.

Editing Tools
üîç
Find & Replace
Press Ctrl+H. Type the word you want to find in the "Find" box, and the replacement word in "Replace with". Click Replace All to change every instance at once ‚Äî saves huge amounts of time in long documents!

Right-click any word ‚Üí Synonyms for a quick list. Or go to Review tab ‚Üí Thesaurus for a full panel. Use it to avoid repeating the same word and to find stronger vocabulary. Great for essays!

Ctrl+Z Undo ‚Äî reverses your last action. Press multiple times to go further back.
